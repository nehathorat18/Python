# Linux Commands

## Process Management

- `ps` ā†’ check running processes  
- `ps -a` ā†’ show processes of all users 
- `ps aux` ā†’ detailed list of all processes  
- `ps aux | grep name` ā†’ find specific process  
- `top` ā†’ live process monitoring (CPU, memory, PID)  
- `kill ` ā†’ terminate a process  

---

## File System & User Management

- `useradd username` ā†’ create new user  
- `userdel username` ā†’ delete user  
- `groupadd groupname` ā†’ create new group  
- `usermod -aG group user` ā†’ add user to group  

- `chmod` ā†’ change file permissions  
- `chown` ā†’ change file owner  
- `chgrp` ā†’ change group ownership  

---

## Networking Troubleshooting

- `nslookup ` ā†’ get IP address of domain  
- `ping ` ā†’ check connectivity  
- `dig ` ā†’ detailed DNS info  
- `curl` ā†’ transfer data or to upload data 
- `wget` ā†’ download file  
- `traceroute ` ā†’ track network path
